# ai-expense-tracker
app link - https://ai-expense-tracker-npnsiwysd2p4btvofrbtzx.streamlit.app/
AI Smart Expense Tracker for Teenagers is a Streamlit-based web app that helps users record daily expenses, automatically detect categories like food, travel, and shopping, visualize spending with colorful charts, and view monthly summaries. It also predicts next month’s spending using machine learning for smarter financial planning.
